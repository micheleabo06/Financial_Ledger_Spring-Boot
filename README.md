This is a small banking application I built to learn how Spring Boot connects a website to a database. It simulates a basic bank where you can store account information and move money between people safely.

How I Built It: 

    Language: Java 25 (using the latest features I'm learning in class).

    Framework: Spring Boot (this handles the "plumbing" of the app).

    Database: H2 (a temporary database that lives in the computer's memory while the app is running).

    Logic: I used a Service Layer to make sure the math for the money transfers happens correctly every time.

Other things I've learned from this project:

    CRUD Operations: I learned how to Create, Read, Update, and Delete data in a database.

    Automatic Math: The app automatically checks if you have enough money before letting a transfer go through.

    Problem Solving: I had to fix several "404 Not Found" errors and port conflicts while setting this up.
